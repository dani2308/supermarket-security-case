# supermarket-security-case

## Overview
This repository presents a complete cybersecurity case study based on an academic project, covering both the security assessment phase and a subsequent incident response scenario in a supermarket enterprise environment.

## Scope
- Risk and environment assessment
- Threat and vulnerability analysis
- Incident response and mitigation
- Lessons learned and security improvements

## Project Phases

### Phase 1 — Security Assessment & Risk Analysis
- Infrastructure and business context analysis
- Identification of security weaknesses
- Threat modeling and risk assessment

### Phase 2 — Incident Response
- Incident identification and impact analysis
- Response and containment strategy
- Security improvement recommendations

## Key Takeaways
- Importance of proactive security planning
- Impact of insufficient monitoring and controls
- Value of structured incident response procedures

## Notes
This project is based on academic work. All data has been anonymized and adapted for educational purposes.
